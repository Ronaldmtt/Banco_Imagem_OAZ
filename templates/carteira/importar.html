Arquivos Excel

Para arquivos Excel, o sistema reconhece automaticamente as colunas:

  • REFERÊNCIA E COR → SKU
  • NOME → Descrição
  • NOME / COR → Cor
  • GRUPO → Categoria
  • SUBGRUPO → Subcategoria
  • ENTRADA → Coleção
  • ESTILISTA → Estilista
  • QUANDO → Shooting
  • FOTO → Status Foto

Arquivos CSV

Para arquivos CSV simples:

sku, descricao, cor, categoria

Dicas

  • Arquivos Excel com múltiplas abas: selecione a aba desejada
  • SKUs duplicados serão ignorados
  • O status "SIM" na coluna FOTO marca como "Com Foto"
  • Cruzamento automático com produtos cadastrados
