Arquivos Excel

O sistema reconhece automaticamente as seguintes colunas (ignora maiusculas/minusculas e acentos):

  • REFERENCIA E COR ou SKU ou CODIGO (obrigatorio)
  • NOME → Descricao
  • NOME / COR ou COR → Cor
  • GRUPO ou CATEGORIA → Categoria
  • SUBGRUPO → Subcategoria
  • ENTRADA → Colecao
  • ESTILISTA → Estilista
  • QUANDO → Shooting
  • FOTO → Status (SIM/NAO)

Colunas nao reconhecidas serao ignoradas.

Arquivos CSV

Para arquivos CSV, use as mesmas colunas:

SKU, DESCRICAO, COR, CATEGORIA, QUANTIDADE

Dicas

  • Excel com multiplas abas: escolha uma ou importe todas
  • SKUs duplicados serao ignorados (nao duplica)
  • Coluna FOTO com "SIM" marca como "Com Foto"
  • Cruzamento automatico com produtos cadastrados
  • Quantidade padrao = 1 se nao informada
